# dry-fruits-selling-website
website : 
This websites deals with the online trading and retailing of dryfruits and provide a secure and easy way to shop. In this project we have made a very descriptive and pictorial representation of various dryfuit products. It also consists of remove options so that if anyone by mistake added any item to cart can easily remove it.It also consists of a payment gateway which consists of internet banking, pay by wallet, pay by paytm and also by using debit card. This project in future can be very helpful for people as well as for society.
